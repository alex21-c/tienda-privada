❓ Cómo usar este panel

📝 Para agregar un producto:

  1. Completa la información básica (nombre, precio, origen, etc.)
  2. Agrega colores usando el botón "+ Agregar Color"
  3. Para cada color, puedes agregar tallas con precios específicos
  4. Haz clic en "Guardar Producto (Auto)" para guardar automáticamente
  5. O usa "Generar JSON (Manual)" para copiar y pegar manualmente

📋 Formato de la hoja "productos":

A: id | B: nombre | C: precio | D: origen | E: imagen | F: activo (SI) | G: descripcion | H: tiene_variantes (SI) | I: variantes | J: enlace | K: categoria_genero | L: colores_nombres | M: colores_imagenes | N: colores_precios
